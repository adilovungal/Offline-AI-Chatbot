# Offline AI Chatbot

A fully offline AI chatbot built using Flask and an open-source language model (TinyLLaMA) running locally via Ollama.

## Features
- Runs completely offline
- Flask backend
- Simple web-based chat UI
- Uses open-source LLMs (no paid APIs)

## Tech Stack
- Python
- Flask
- HTML/CSS/JavaScript
- Ollama + TinyLLaMA

## How to Run
1. Install Ollama
2. Pull the model: ollama pull tinyllama
3. Install dependencies:pip install -r requirements.txt
4. Run the app: python app.py
5. Open Browser: http://127.0.0.1:5000
